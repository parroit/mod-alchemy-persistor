Status and roadmap

The project is actually tested on Sqlite. Other Sql Alchemy dialects are still to be tried.

I’ve not yet submitted the module to the official vert.x repository, so you’ll have to use my personal fork of the repository in order to try the module.

Before submission, I want to try the project at least with mysql and mssql dialects.

Moreover, I want to add more test and some sample of use.

You could find on the issues page on github a complete list of all things I want to do before submission. I’ll submit once all issue for milestone 1.0 will be successfully closed.

The project is not yet ready for production use, you’ll be warned! However, you’re welcome to try it and to help improve it.

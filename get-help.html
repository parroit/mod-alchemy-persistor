Help each other!

How to get help

To ask for help, please post a question On stackoverflow. I’ll be alerted via mail of your questions, and I’ll do my best to respond there as soon as possible.

If you prefer, feel free to contact me directly via mail

If you discover some bugs, post an issue to github repo. If possible, please include an SSCCE

How to give help

Any help is higly appreciated. You can:

  1. write more example on the module usage
  2. write more test
  3. solve issues
  4. improve this site
  5. implements new features

To do any of these, simply fork the repo and request a pull when you’re done.

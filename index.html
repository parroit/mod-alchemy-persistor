Welcome to Alchemy Persistor web site

Abstract

Alchemy Persistor is a module for the Vert.x web framework that allow you to persist your data models using Python Sql Alchemy.

This module allows data to be saved, retrieved, searched for, and deleted in a Sql Alchemy managed database.

It is inspired from mongo persistor, which I use as a starting skeleton for the project, and from which I take the format for all json command.

To get started, see Getting started

Getting started

1) Install the module

Thanks to the modules engine of Vert.x, there is no need to install anything to use this module, just deploy a module giving it a name of “eban.alchemy-persistor-v0.1” and a JSON configuration as explained in documentation.

However, since the module has not yet been submitted to the official vert.x repository, you have to specify to use my repo fork when running your vertx verticles.

You can do this adding the option -repo http://github.com/parroit when you run vertx.

2) Read the documentation

You can read the documentation of the module. Syntax of JSON configuration and commands is explained there.

3) Start play with it

To start playing with the module, you can look at the examples source code or at the tests to understand the usage.

If you write some small example, and want to contribute, please send me them and I’ll include them in the package.

4) Contribute!

If you want to help the project go on, see what you can do
